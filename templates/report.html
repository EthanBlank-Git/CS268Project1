Expectations


Style

  • 1. The website was created for a client that wasn't either of us.
  • 2. We have a header on every page of the website that includes the title of the current page and the button for the navbar.
  • 3. Our website has 11 separate pages that each have their own unique purpose.
  • 4. Our site has navigation links that takes you between the pages and to some outside resources.
  • 5. Our page uses media queries for collapsing content in a vertical column.
  • 6. We used similar font styles and layouts throughout our project.
  • 7. Yea have images in most of the pages with alt attributes.
  • 8. We have a favicon on top of the navbar.
  • 9. We utilized a images, CSS, and JS folders for organization.
  • 10. We used some Bootstrap with a couple of the page elements.
  • 11. We used flask so this isn't valid.
  • 12. All of our files contain lowercase characters with some having numbers.
  • 13. We centered a lot of our content and used aligning css to make our content aligned.
  • 14. If you look at our css file then you will see that we had multiple fonts faces.
  • 15. We have 7 pages that have forms implemented on them.
  • 16. We have links to eternal resources on a couple pages some leading to products and others to references.
  • 17. Our code is readable and we used meaningful names for classes and IDs.
  • 18. We have some animations with hovering over some pictures and a collapsable navbar.
  • 19. We utilized SQLite as our database.
  • 20. We have login functionality however still need to make it different.
  • 21. Project 1 remarks will be addressed below.
